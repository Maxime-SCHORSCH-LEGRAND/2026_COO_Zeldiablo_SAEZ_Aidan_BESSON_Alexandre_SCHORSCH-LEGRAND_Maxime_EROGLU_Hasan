# Fonctionnalités prévues version 1

- fonctionnalité 1 : Déplacement et affichage du Héros
  - descriptif texte : Le héros est créé dans le labyrinthe et le joueur peut le faire bouger à l'aide des commandes de direction. Le moteur graphique affiche le labyrinthe et le héros à l'écran.
  - critères de validation :
    - Le héros se déplace correctement d'une case à chaque pression de touche.
    - Le héros ne peut pas traverser les murs du labyrinthe.
